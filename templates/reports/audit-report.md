### Appendix H — Glossary of Technical Terms

| Term | Definition |
|---|---|
| Advanced Persistent Threat (APT) | A sophisticated, long-term cyberattack in which an adversary gains unauthorised access to a network and remains undetected for an extended period |
| Attack Surface | The sum of all potential points where an adversary could attempt to enter a system or extract data |
| Business Impact Analysis (BIA) | A process to determine and evaluate the potential effects of an interruption to critical business operations |
| Common Vulnerability Scoring System (CVSS) | An industry standard for assessing the severity of computer system security vulnerabilities |
| Defence-in-Depth | A security strategy employing multiple layers of security controls to protect assets |
| Endpoint Detection and Response (EDR) | Security technology that continuously monitors endpoints to detect and respond to advanced threats |
| Indicators of Compromise (IoC) | Forensic artefacts observed in a network or operating system that indicate a security breach |
| Lateral Movement | Techniques used by adversaries to progressively move through a network after initial compromise |
| Least Privilege | The principle of granting users and systems only the minimum access required to perform their functions |
| National Critical Information Infrastructure (NCII) | Infrastructure designated by NACSA as critical to national security, public order, safety, economy, or essential services |
| Penetration Testing | Authorised simulated cyberattacks to evaluate the security of a system |
| Phishing | Social engineering attacks using deceptive electronic communications to obtain sensitive information |
| Ransomware | Malicious software that encrypts a victim's files and demands payment for decryption |
| Security Posture | The overall cyber security strength and resilience of an organisation |
| Threat Intelligence | Evidence-based knowledge about existing or emerging threats to assets |
| Zero-Day Vulnerability | A software vulnerability unknown to the vendor or lacking a patch at the time of exploitation |
